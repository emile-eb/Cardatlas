# CardLens Prototype

Front-end iPhone-first prototype for baseball card scanning and collection tracking.

## Run on Windows (Expo)

1. `npm install`
2. `npm run start` (mobile) or `npm run web` (browser)
3. In Expo Dev Tools:
   - press `w` for web
   - press `i` for iOS simulator (if available via remote/mac tooling)
   - or scan QR in Expo Go for device testing

## Stack

- Expo + React Native + TypeScript
- Expo Router navigation
- Expo Google Fonts (Inter + Space Grotesk)
- Local mock data + local state only
